Documentation for using Grunt & Bower to compile, lint, minify and download dependencies.

Install

  1. Install Node.js

    Use the install package on the website http://nodejs.org/, or via command line http://howtonode.org/how-to-install-nodejs

  2. Install Grunt.js

    Install Grunt Command Line Interface (Grunt CLI) allowing you to execute grunt on the command line.

    npm install -g grunt-cli

  3. Install ImageMagick (image minification)

    Prerequisite: install Homebrew: http://brew.sh/

    Run command: brew install ImageMagick

    Error:

    The `brew link` step did not complete successfully
The formula built, but is not symlinked into /usr/local
Could not symlink include/libltdl/usr/local/include is not writable.

    Solution:

    Go to the directory cd /usr/local/include and open permissions on the folder sudo chmod -R 777 . Run these commands to esure proper linking

    brew link imagemagick
brew link jpeg
brew link libtool
brew link libpng
brew link freetype

    If grunt build.images.web throws an error still, you may need ghostscript. brew install ghostscript

  4. Install svg-sprite globally

    Used for svg spriting (https://github.com/jkphl/svg-sprite)... grunt build.sprite

    npm install svg-sprite -g

Project Install

  1. Go to the terminal and navigate to your project folder.
    $ cd FOLDER/LOCATION/CONTAINING/GRUNTFILE.JS

  2. Install project specific gems from the Gemfile.lock file.

    bundle install

  3. Install all necessary packages inside package.json into a node_modules folder.

    npm install

  4. Run initial tasks again to gaurd against error when running this through Yeoman.

    grunt build.dev

  5. Now you should be able to run grunt watch to compile css among other commands found in the Quick Reference section.
